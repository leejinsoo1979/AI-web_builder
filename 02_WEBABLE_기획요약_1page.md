# WEBABLE 기획요약 1page

## 제품 정의

WEBABLE은 코딩 없이 웹사이트, 쇼핑몰, 예약 사이트, 콘텐츠 허브를 만들고 운영할 수 있는 AI 네이티브 SaaS형 웹사이트 빌더입니다. 사용자는 브랜드/업종/타깃/톤을 입력하면 AI가 사이트명, URL 후보, 메뉴 구조, 페이지별 섹션, 카피, 디자인 무드, SEO 초안을 생성하고, 운영자는 섹션 기반 빌더에서 이를 편집해 즉시 게시합니다.

## 차별화 방향

기존 웹빌더는 템플릿과 수동 편집 중심입니다. WEBABLE은 AI가 사이트 제작뿐 아니라 상품 등록, 주문 이슈 요약, 고객 세그먼트, 캠페인 문구, 성과 분석, SEO/AEO/GEO 개선까지 돕는 “브랜드 운영 에이전트”를 목표로 합니다. AI의 모든 작업은 권한 기반으로 제한되고, 변경 diff와 운영자 승인 후 적용됩니다.

## 핵심 모듈

- SaaS: 조직, 사용자, 멤버십, 사이트, 요금제, AI 크레딧, 권한, 감사 로그
- Builder: 섹션/블록 기반 편집, 드래그 앤 드롭, 테마, 템플릿, 반응형, 게시/롤백
- CMS: 페이지, 메뉴, 게시판, 블로그, 갤러리, 입력폼, 지도, 코드 위젯
- Commerce: 상품, 옵션, 재고, 장바구니, 주문, 결제, 배송, 쿠폰, 리뷰/문의, 환불/반품
- Booking: 서비스/공간/객실/클래스 예약, 캘린더, 정책, 관리자 예약
- CRM/Marketing: 고객, 세그먼트, 이벤트, 자동 메시지, 알림톡/SMS/이메일, 광고/픽셀
- AI: 사이트 생성, 자연어 편집, 콘텐츠/상품/주문/CRM/분석/지원 에이전트
- Ops: 도메인, SSL, CDN, 백업, 로그, 모니터링, 보안/개인정보 설정

## MVP 목표

MVP는 “가입 - AI 사이트 초안 생성 - 섹션 편집 - 미리보기 - 도메인/SSL - 게시 - 롤백” 흐름을 완성하는 데 집중합니다. 쇼핑몰 전체 기능은 V1에서, CRM/예약/고급 AI 운영 에이전트는 V1.5-V2에서 확장합니다.

## 권장 기술 방향

Frontend는 Next.js/React/TypeScript, Backend는 NestJS 또는 Spring Boot, DB는 PostgreSQL + RLS, Cache/Queue는 Redis/BullMQ 또는 SQS, Storage는 S3 호환 스토리지, AI는 모델 게이트웨이/프롬프트 레지스트리/툴 런타임/평가 파이프라인으로 구성합니다.

## 성공 지표

North Star는 월간 게시 활성 사이트 수입니다. 보조 지표는 가입 후 24시간 내 첫 게시율, AI 제안 적용률, 첫 게시까지 걸린 시간, 유료 전환율, 쇼핑 GMV, AI 비용/매출 비율, 공개 사이트 LCP, API p95 latency, churn입니다.
