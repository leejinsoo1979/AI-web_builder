# WEBABLE 개발 기획 패키지 v1.0

WEBABLE은 아임웹급 웹사이트/쇼핑몰/예약/CRM 빌더 기능에 AI 자동 생성과 운영 에이전트를 결합한 SaaS 제품입니다. 이 ZIP은 개발 착수 전에 팀이 공유해야 할 제품 범위, 기능 정의, 기술 아키텍처, 데이터 모델, API 초안, AI 스키마, 보안/컴플라이언스 체크리스트를 포함합니다.

## 사용 순서

1. `01_WEBABLE_완벽_개발기획안_v1.0.pdf`로 전체 방향을 빠르게 검토합니다.
2. `03_WEBABLE_기능정의_로드맵_백로그.xlsx`에서 MVP/P0 기능만 필터링해 1차 스프린트 범위를 확정합니다.
3. `08_WEBABLE_AI_SiteSpec_schema.json`과 `05_WEBABLE_API_OpenAPI.yaml`을 기준으로 프론트엔드/백엔드/AI 팀의 데이터 계약을 맞춥니다.
4. 개발 착수 전 `09_WEBABLE_보안_컴플라이언스_체크리스트.md`를 기준으로 개인정보/AI/전자상거래/권한 리스크를 점검합니다.

## 핵심 전제

- 아임웹과 동일한 기능 범위를 목표로 할 수는 있지만, 화면 구성, 문구, 템플릿, 브랜드, UI 흐름은 독자적으로 설계해야 합니다.
- 초기 MVP는 완전 자유형 캔버스보다 섹션/블록 기반 빌더로 시작하는 것이 안정적입니다.
- AI는 바로 게시하지 않고 변경 diff, 검증, 운영자 승인, 롤백을 기본으로 합니다.
- SaaS형 제품이므로 멀티테넌트 격리, 권한, 감사 로그, 개인정보 처리방침, AI 데이터 사용 정책이 제품 초기부터 필요합니다.
